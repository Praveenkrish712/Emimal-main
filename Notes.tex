CSS - Cascading style sheet

1) Types of CSS
i - inline
ii - internal
iii - external
inline > internal > external

2) CSS Syntax
selector{
    property: value;
}

3) Types of selector
i - Element selector -  (h1) (h2) (p)
ii - Class selector - (.) - multiple times can use same class name
iii - Id selector - (#) - unique - only one time can use the same id name
iv - universal selector - (*) -  to be apply with entire site
v - Group selector - class or id name contains different tags we want to change to same style means we go with Group selector

4) flex-wrap - nowrap, wrap, wrap-reverse
5) Align-content -(Y) (Must be set with height) (only wrap will enable then only it will work) - start, center, end, space-around, space-evenly, space-between
